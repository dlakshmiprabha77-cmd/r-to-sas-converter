# R to SAS Clinical Converter

## 📌 Project Overview
This project demonstrates conversion of R code to SAS code for clinical trial datasets using CDISC SDTM standards.

## 📊 Features
- Create SDTM datasets (DM, AE)
- R to SAS code mapping
- Clinical trial data handling

## 🛠️ Technologies Used
- R Programming
- SAS
- CDISC SDTM

## 📂 Project Structure
- R_code/ → R scripts
- SAS_code/ → SAS scripts
- sample_data/ → Input datasets

## 🔄 Example Mapping

| R | SAS |
|--|--|
| paste0() | cats() |
| toupper() | upcase() |
| data.frame() | data step |
